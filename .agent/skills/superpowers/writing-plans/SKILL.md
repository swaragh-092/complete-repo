# Writing Plans Skill

## Goal

Take a validated design and outline tasks that can be implemented step-by-step.

## Instructions

1. Read design spec.
2. Divide into tasks (2–5 minutes each).
3. Add exact paths and tests for each task.

## Output

- Task list with acceptance criteria.
