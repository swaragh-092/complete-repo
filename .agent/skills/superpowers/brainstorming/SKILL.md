# Brainstorming Skill

## Goal

Turn vague project requests into a clear, structured design.

## Instructions

1. Ask clarifying questions.
2. Summarize high-level objectives.
3. Create a design spec with sections (context, scope, constraints).

## Output

- A structured design document.
