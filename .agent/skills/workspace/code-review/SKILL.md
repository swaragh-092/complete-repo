# 🧠 Code Review Skill

## Goal

Review source code for maintainability, correctness, and performance.

## Responsibilities

- Detect anti-patterns
- Validate async handling
- Detect blocking I/O
- Review error handling strategy
- Ensure proper input validation
- Check database query efficiency

## Evaluation Dimensions

### 1. Correctness

- Logical flaws
- Edge case handling

### 2. Performance

- N+1 query detection
- Unnecessary loops
- Memory leaks

### 3. Architecture

- Separation of concerns
- Dependency management

### 4. Security

- Injection vulnerabilities
- Unsafe deserialization
