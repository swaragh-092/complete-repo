What is the Super Administrator Module?

The Super Administrator Module is the core administrative system of a hybrid multi-tenant platform. It provides centralized control for managing all tenants (organizations), but also exposes selected functionality to organization-level (tenant) administrators for operations like viewing subscription details, renewing plans, and accessing invoices.
